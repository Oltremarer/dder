## v5 Pass/Fail Rules

Density gate:

- `rare_useless` unique `obs_action_next >= 50` per seed
- `rare_valuable_zero_precursor` unique `obs_action_next >= 50` per seed
- `rare_useless` mean multiplicity <= 3x precursor mean multiplicity
- `rare_useless` median distance to common-zero > 0 for at least 2/3 seeds

H1:

- kNN Recall@Top10 rare valuable >= 0.50 all seeds, or
- kNN Recall@Top20 rare valuable >= 0.70 all seeds

H2:

- exists one ratio in `{0.005, 0.01, 0.02, 0.05, 0.10}`
- all seeds have `rare_useless_count >= 10`
- all seeds have `rare_useless_fraction >= 0.04`
- all seeds have `rare_useless_enrichment >= 2.0`
- all seeds have `rare_valuable_all_count >= 50` or `rare_valuable_recall >= 0.30`

H3:

- R2V selected rare-useless fraction < candidate rare-useless fraction all seeds
- R2V selects nonzero zero-reward precursors all seeds
- R2V positive reward fraction <= 0.50 all seeds
- R2V zero-precursor recall > reward-only zero-precursor recall

H4:

- optional-invalid excluded from H1-H3
- invalid-risk AUROC >= 0.85 all seeds

If v5 fails any of density/H1/H2/H3 after one clean implementation, stop the synthetic-grid Level 0 route.
