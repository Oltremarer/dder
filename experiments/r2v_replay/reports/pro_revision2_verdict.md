## Required Dataset Revision

Add a new config instead of overwriting the existing one:

- `experiments/r2v_replay/configs/level0_sparse_grid_distractor_v2.yaml`

Core changes:

- grid size: 21 or 23
- distractor pockets: 6-8 scattered off-route pockets
- each pocket: 3-5 cells with a narrow doorway
- distractor probe policy ratio: 0.03-0.05
- rare_useless label scope: inside deep pocket only
- path-to-decoy transitions remain `common_zero` unless inside a deep pocket

Behavior policy:

- `rare_distractor_probe_policy`
- choose one distractor pocket uniformly
- navigate noisily to the pocket
- enter pocket and take 1-4 random actions
- reward stays 0
- transitions must be real environment actions, not teleport or invalid samples

Label rule:

`rare_useless` iff:

- real transition
- reward == 0
- episode_return == 0
- state or next_state is inside a deep distractor pocket
- not on the main corridor
- not within success precursor window
- progress_to_goal <= 0

Do not label ordinary route-to-decoy transitions, random wandering near start, common off-path motion, or optional invalid stress samples as `rare_useless`.
